# 定制化 WujieDemo

## 快速开始

:::tip
wujie 可以通过脚手架来定制需要的主子应用框架的Demo，以及其他特性
:::

### 安装 create-wujie

```js
npx create-wujie@latest
```

### 支持包管理器

WujieDemo 采用 TurboRepo 进行管理 支持多种包管理器

- npm
- pnpm
- yarn

### 支持路由定制

可以选择主应用跟子应用的不同路由模式 `hash | history`
